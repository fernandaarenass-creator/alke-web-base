# Alke Web Base - Módulo 6 Django
Este proyecto es una aplicación web básica desarrollada con el framework Django para **Alke Solutions**.

## Características:
- Arquitectura MVT.
- Páginas dinámicas.
- Uso de archivos estáticos (CSS).
- Configuración de entornos virtuales.

## Cómo ejecutar:
1. Clonar el repo.
2. Crear venv: `python -m venv venv`.
3. Activar y ejecutar: `python manage.py runserver`.
